## 🌐 Option 1: Deploy to GitHub Pages (FREE)

### Step 1: Create GitHub Repository

```bash
cd portfolio
git init
git add .
git commit -m "Initial portfolio commit"
```

### Step 2: Push to GitHub

```bash
git remote add origin https://github.com/aishwaryasingh0217-collab/portfolio.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages

1. Go to: https://github.com/aishwaryasingh0217-collab/portfolio
2. Click **Settings** → **Pages**
3. Under "Source", select **main** branch
4. Click **Save**
5. Your site will be live at: `https://aishwaryasingh0217-collab.github.io/portfolio/`

---

## 🚀 Option 2: Deploy to Vercel (FREE)

### Method A: Using Vercel CLI

```bash
cd portfolio
vercel --prod
```

Follow the prompts and your site will be live!

### Method B: Using Vercel Dashboard

1. Go to: https://vercel.com/aishwaryasingh0217-collabs-projects
2. Click **Add New** → **Project**
3. Import your GitHub repository
4. Click **Deploy**

Your site will be live at: `https://portfolio-[random].vercel.app`

---

## 🎨 Option 3: Deploy to Netlify (FREE)

### Drag & Drop Method:

1. Go to: https://app.netlify.com/drop
2. Drag the entire `portfolio` folder
3. Done! You get a free URL

### GitHub Method:

1. Push to GitHub (see Option 1)
2. Go to: https://app.netlify.com
3. Click **Add new site** → **Import from Git**
4. Select your repository
5. Click **Deploy**
