# Aishwarya's Portfolio Website

## 🎨 Features

- **Soft, Happy Colors**: Eye-pleasing pastel colors (pink, blue, peach, lavender)
- **Animated Cherry Blossoms**: Beautiful falling petals with visible trees
- **3D Animated Running Dog**: Cute dog with moving legs, wagging tail in navbar
- **Horizontal Scrolling**: Unique side-scrolling navigation
- **Day/Night Mode**: Toggle between light and dark themes
- **WhatsApp + Email Integration**: Floating buttons with pre-filled messages
- **Fully Responsive**: Works on Android and iOS (all screen sizes)

## 🚀 How to View

1. **Double-click** `index.html` to open in your browser
2. Use **arrow keys** (← →) or **mouse wheel** to navigate between sections
3. Click the **dots** in the navbar to jump to specific sections
4. On mobile, **swipe left/right** to navigate
5. Click **sun/moon icon** to toggle dark mode
